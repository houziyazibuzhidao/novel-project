# 《我在03年炒凶宅》第二章细纲

## 第二章：搬家

### 核心目标
- 从"得意"滑入"不安"——秦磊搬进新房，一切开始不对劲
- 铺设凶宅的第一个层次：日常中的微末异样
- 邻居爆料：老太太死了半个月才被发现——真鬼设定的第一块实锤
- 章末钩子：第一件无法解释的事发生

### 叙事视角
第一人称，秦磊视角，从兴奋到疑惑到有点发毛

### 字数
2500-3000字

---

### 场景拆解（4个节拍）

**节拍1：搬家（约500字）**

场景：周末搬进新房
- 秦磊一个人搬家，大勇帮忙（大勇还没买房，此时还没有婚房问题）
- 东西不多：一个蛇皮袋的衣物、一箱泡面、一床被子、几件厨具
- 2003年生活质感：出租车拉家具、超市买便宜床单、暖壶、脸盆
- 秦磊收拾完站在客厅，很满足——终于有自己的房子了
- 大勇帮忙完走的时候，说了一句："这楼道怎么这么大味儿？"指的是蚊香味，但当时没人当回事

**情绪：** 兴奋、满足、终于有了自己的窝

---

**节拍2：第一夜——刚好不对（约600字）**

场景：搬进去当夜
- 秦磊一个人睡，关灯之前一切正常
- 几个"刚好不对"的小细节，不是吓人，是让细心的读者心里一动：
  - 窗户关不严，他以为是老房子变形
  - 水龙头滴水，拧了两下还在滴，垫了块抹布
  - 整层楼特别安静，好像只有他一个人住
- 他给大勇发短信炫耀（2003年发短信，不是微信），大勇回："便宜没好货哈哈"
- 他笑着关机睡觉

**情绪：** 平静中的微末异样，秦磊完全不在意

---

**节拍3：邻居老大爷（约800字）**

场景：第二天早上，秦磊出门上班
- 楼道里蚊香味很浓——不是一家在点，是好几层都在点
- 下楼碰到六楼对门的老大爷，七十多岁，坐在门口马扎上
- 老大爷主动搭话："新搬来的？"
- 秦磊说是，买了601
- 老大爷的表情变了，欲言又止，最后说了句："你一个人住？"
- 秦磊说是
- 老大爷压低声音："你那屋之前住的老太太，姓李……走了半个月，才被人发现。"
- 秦磊愣了一下，说："哦。"
- 老大爷又看了他一眼："楼里夏天蚊子多，大家点蚊香……也不全是蚊子的事。"
- 秦磊没听懂，也没多想，"嗯"了一声就走了

**情绪：** 微妙的不安，但秦磊还在给自己找理由——老人嘛，爱瞎想

关键细节：
- "走了半个月才被发现"这句话要轻描淡写地说出来，老大爷不是故意吓人，是随口提的
- 蚊香味的真正含义在这里埋下——不是防蚊子，是压味道
- 老大爷的"也不全是蚊子的事"是第二次暗示，但秦磊没接住

---

**节拍4：第一件怪事（约600字）**

场景：当晚
- 秦磊下班回来，做饭、吃饭、看电视，一切正常
- 睡到半夜，被声音吵醒
- 哗啦哗啦——水龙头开了
- 他爬起来去厨房，水龙头大开着，水哗哗地流
- 他关上，擦了擦手，心想：下午明明拧紧了
- 回床上躺下，翻了个身
- 突然想起老大爷白天说的那句"走了半个月才被发现"
- 他睁开眼，盯着天花板
- 厨房里，水龙头又开始滴了——滴答、滴答、滴答

**结尾句：**
- 不要写他害怕，写他睡不着
- "那天晚上我翻来覆去到天亮，总觉得厨房那边有什么声音，但每次竖起耳朵去听，又什么都没有。"
- 钩子：水龙头到底是自己开的，还是有人开的？

**情绪：** 不安+自我怀疑，还没到害怕的程度

---

### 本章不出现的内容
- 残卷、何三七、瞎子二婶
- 任何驱邪知识
- 大勇的婚房问题
- 秦磊的爷爷（第三章才想起爷爷）
- 直接的恐怖画面——只有"不对"，没有"吓人"

### 去AI味检查清单
- [ ] 不用"令人毛骨悚然""不寒而栗""脊背发凉""毛骨悚然""诡异"
- [ ] 恐怖靠细节和留白，不靠形容词堆砌
- [ ] 口语感：允许用"他妈的""我擦"这类口语
- [ ] 秦磊的声音要像24岁男生，不是文学青年
- [ ] 2003年细节要实：发短信不是发微信、诺基亚、蛇皮袋搬家
- [ ] 每个异常事件之间有日常——上班、吃饭、做饭，恐怖是从日常里渗出来的
- [ ] 老大爷的话要像老人说话，啰嗦、含蓄、意有所指但不明说
- [ ] 水龙头开了不是"鬼开了水龙头"，是秦磊的视角：我关了，它又开了

### 节奏波形
```
兴奋满足 → 平静微异 → 微妙不安 → 睡不着
  ▔▔▔▔      ▔▔▔▁▁      ▔▁▁▁      ▁▁▁▁
```
