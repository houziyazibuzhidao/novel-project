# 《我在03年炒凶宅》第一凶宅arc·3章版 细纲

## 第一章：五万二

**功能**：买房→入住→第一个异样信号→章末钩子
**字数目标**：3000-3500字
**视角**：秦磊第一人称

### 节拍

1. **看房砍价**（~800字）
   - 03年小城，秦磊24岁，超市理货员，攒了四年钱
   - 周哥带看老小区六楼，秦磊嫌贵，周哥降价到五万二
   - 秦磊心里算账：月供没有，一次性付清，比租房强
   - 关键细节：钥匙扣上有个旧铜钱（伏笔），进门茶几上白瓷碗插着三根烧剩的香梗，门框上贴着褪色红纸条
   - 给妈打电话借四千，"我妈在镇上给人缝扣子，一件两毛钱"
   - 付钱拿钥匙

2. **一个人搬进去**（~1000字）
   - 没叫人，自己扛着蛇皮袋爬六楼
   - 去超市买东西：19块9床单、8块5暖壶、3块5脸盆——金牛座精打细算
   - 收拾屋子，擦灰，铺床，看着自己的房子，觉得值
   - 窗户关不严，窗框和窗台之间总有一条缝

3. **老大爷爆料**（~600字）
   - 第二天出门，楼道里遇到邻居老大爷
   - 老大爷认出他是新搬来的，问他住哪户
   - "你那屋之前住的老太太？死了半个月才被人发现。"
   - 秦磊心里咯噔一下，但嘴上没事人似的
   - 老大爷又说"那阵子楼道里总有一股子味，也不全是蚊子的事……后来物业天天点蚊香才压下去"
   - 秦磊回家路过茶几，看了一眼那三根香梗——原来不是供的，是压味儿的

4. **第一晚**（~800字）
   - 半夜被声音弄醒，厨房水龙头在滴水
   - 去厨房关了，回来躺下
   - 过了一会儿，又滴
   - 起来又关，这次拧紧了
   - 躺回床上，盯着天花板
   - 关灯前明明关紧了——**章末钩子**

## 第二章：不对劲

**功能**：恐怖加速→门缝光→想起爷爷
**字数目标**：3000-3500字
**视角**：秦磊第一人称

### 节拍

1. **异事叠加**（~1200字）
   - 搬进去第五天，开始不对劲
   - 早上醒来发现自己睡在地上，被子在床上，人是被"推"下去的
   - 厨房铁锅盖子位置不对，像被人动过
   - 香梗排列变了，跟之前不一样了
   - 红纸条掉了一角
   - 每件单独都能解释，但凑一起让人别扭
   - 秦磊给自己找理由：睡相差、记错了、胶水干了

2. **敲门**（~800字）
   - 第七天半夜，卧室门被敲响
   - 咚咚咚，三下，节奏不快不慢
   - 秦磊先想大勇，转念——大勇这会儿早睡了
   - 摁亮手机，走到门口，手放门把手上
   - 没开。站了几秒，外面没声音了
   - 从那以后睡觉不关门，开着灯，电视声音调到最小
   - 但——睡前明明把电视关了，半夜又自己亮了**章内推进**

3. **门缝光**（~1000字）
   - 又一晚，躺在床上，看到门缝底下的光线
   - 客厅灯自己开了
   - 然后光线突然没了——有什么东西挡住了门缝
   - 秦磊绷紧了，想冲出去但腿不听使唤
   - 数秒，等光回来
   - 光回来了，那个东西走了
   - 天亮后坐在沙发上，无法解释
   - 突然想起爷爷——他好像说过什么，关于房子的，关于死过人的房子的。但想不起来了**章末钩子**

## 第三章：爷爷

**功能**：回忆→执行仪式→蜡烛灭→恢复正常→大勇求救电话
**字数目标**：3000-3500字
**视角**：秦磊第一人称

### 节拍

1. **拼命想**（~300字）
   - 上班想，下班想，脑子里就一件事
   - 爷爷走了两年了，问不了
   - 闭上眼使劲回忆——碎片开始往外蹦

2. **爷爷的回忆**（~1200字）
   - 小学三四年级以前，暑假回老家
   - 爷爷喝了点酒，说了村里的事——谁家老人走得不安稳，谁家搬新宅没打招呼
   - 秦磊当时当鬼故事听，吓得够呛，偏要问："要是已经住进去了怎么办？"
   - 爷爷："人死了不跟人走了？不送，她就不知道自己走了，还当这是她的家。"
   - 仪式三样：一碗清水，一根白蜡烛，一把米
   - "说三遍'你走了，这里不是你的了'。蜡灭了就成了。"
   - "万一蜡不灭呢？"——"那就不是你能管的事了。"
   - 爷爷走到门口回头看了一眼，好像还有话没说

3. **执行仪式**（~1000字）
   - 下班直奔超市：白蜡烛、一碗水（家里有）、一袋米
   - 客厅正中间，摆好，点蜡
   - 秦磊对着空气说了三遍，声音发紧但没怂
   - 等蜡烛灭——火苗晃了几下，灭了
   - 空气变了，秦磊说不上来怎么变的，但就是不一样了，"轻了"

4. **恢复正常**（~500字）
   - 接下来几天一切正常
   - 水龙头没再开，窗户能关严了，电视没再自己亮
   - 门缝光还在，但不是之前那种——没有东西在动了
   - 秦磊觉得自己处理了，有点得意
   - 但夜里还是盯着门缝看，手边放着那根蜡烛
   - "万一蜡不灭"——爷爷没说后面怎么办

5. **大勇的电话**（~400字）**【新】**
   - 某天下班，大勇打来电话
   - 声音不对，压着嗓子："磊子，你……你之前那个房子的事，是真的吧？"
   - 秦磊心里一紧："啥意思？"
   - 大勇："我婚房……我跟我媳妇刚搬进来……"停顿，"她这两天老说听见有人叫她名字。我刚才……我也听见了。"
   - 秦磊握着电话，后背凉了
   - **章末钩子**："你那个仪式……教教我。"

## 伏笔备注

- 旧铜钱：长线伏笔，与残卷/李二叔有关，3章内不回收
- 香梗/红纸条：第1章埋，第2章变化，第3章仪式后正常，已回收
- 窗户关不严：第1章埋，第3章"能关严了"回收
- 电视自己开：第2章埋，第3章"没再自己亮过"回收
- 门缝光：第2章高潮，第3章"没有东西在动了"半回收（光还在，余韵）
- "万一蜡不灭"：长线伏笔，不回收，引向后续
- 爷爷"还有话没说"：引向残卷线
- 大勇婚房：新arc启动钩子
