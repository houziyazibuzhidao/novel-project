# 《我在03年炒凶宅》第四章细纲（大勇婚房Arc开篇）

**功能**：秦磊去大勇婚房→第一次直面"比自己房子更凶的东西"→意识到爷爷的简化仪式可能不够用→章末钩子
**字数目标**：3000-3500字
**视角**：秦磊第一人称

### 节拍

1. **接电话后的犹豫**（~400字）
   - 挂了大勇电话，秦磊站在门口，钥匙还插在锁孔里
   - 心理活动：我那房子的事刚处理完，还没缓过来。但大勇那声音不对——大勇平时大嗓门，压低声音说话的时候就是真怕了
   - 决定去：不是不怕，是不得不去。"四年的上下铺，他打电话给我就是没别人了。"
   - 换衣服出门，骑自行车去城东

2. **大勇婚房：第一印象**（~600字）
   - 城东老小区，比秦磊的小区还旧，楼道灯是坏的
   - 大勇开门，脸色很差，眼窝深陷，跟上次喝酒判若两人
   - 陈晓慧不在家——"她回娘家了，不敢住这"
   - 房子格局：两室一厅，比秦磊的稍大，装修了一半（新房状态：刷了墙漆，但地板还是老地砖）
   - 秦磊注意到几个不对的地方（但不点破）：
     - 卧室门口地上撒了一把米——大勇学他的？但撒得不对，不是围着碗，是沿着门槛撒了一条线
     - 电视插头拔了（跟秦磊一样）
     - 客厅窗户开着，大勇说"不敢关"

3. **大勇讲经过**（~800字）
   - 搬进来第一晚就出事——陈晓慧半夜醒了，听见客厅有人叫她名字，女声，不大不小
   - 大勇以为是隔壁，去客厅看，没人。陈晓慧说声音就在沙发那边
   - 第二晚大勇守着，自己也听见了。不是幻听。"声音像是从墙里传出来的，闷闷的，但很清楚。叫的是'小慧'。"
   - 第三晚：陈晓慧半夜起来上厕所，回来的时候站在卧室门口不动了。大勇问她咋了，她说"床上躺着一个人"。大勇打开灯，床上什么都没有。但陈晓慧坚持说看到一个人影躺在她那侧
   - 大勇学秦磊的法子试了——清水+蜡烛+米。蜡烛灭了。但第二天晚上，声音又来了
   - 大勇的关键台词："你那个法子不管用啊磊子。蜡灭了，我以为好了。结果比之前还凶——她不是叫小慧了，叫的是我。"

4. **秦磊感受异样**（~800字）
   - 天黑了，秦磊没走。说要待一晚看看
   - 大勇在客厅陪他，两人坐沙发，电视关着，灯开着
   - 秦磊开始感觉到——这个房子的东西，跟自己那套不一样
   - 自己601那个，是"不走"——老太太不知道自己死了，还住这儿。送走了就安静了
   - 这个房子，感觉像是"被招来的"——有人叫你名字，你答应了就麻烦了
   - 爷爷从没讲过这种。爷爷讲的都是"没走的"，不是"从外面来的"
   - 夜里两点多，秦磊听到一个声音——不是叫名字，是走廊里的脚步声。一步一步，从大门方向往卧室走
   - 秦磊的恐惧反应：想站起来挡在大勇前面，但腿不听使唤。攥紧拳头，指关节发白
   - 脚步声停在卧室门口。没了。
   - 大勇看着秦磊，脸白得吓人："你听见了吧？"

5. **章末钩子**（~200字）
   - 秦磊心里一沉：自己那个房子的法子，对这个不管用
   - 爷爷说过"万一蜡不灭，那就不是你能管的事"——现在是蜡灭了但没完，比"不灭"还邪门
   - 需要找更懂行的人。秦磊想起爷爷提过一个名字——李二叔。但李二叔早就不在秦家沟了
   - 尾句反抛：秦磊说"明天我先去趟城里，打听个人"——大勇问"谁？"——秦磊没答。他想起爷爷说过的一句话："有些人，你不用找他，他知道你有事自己会来。"
   - 但他没说。

### 写作要点

- **陈晓慧不在场**：回了娘家，只在对话中出现。首次正面描写留给后续章节
- **大勇的恐惧**：不同于秦磊的"愤怒+想冲"，大勇是"硬撑"。膀大腰圆的人害怕的时候不是缩，是撑——说话声音故意大，动作故意粗，但手在抖
- **秦磊的对比感受**：通过秦磊内心独白，建立"601的东西"vs"大勇家的东西"的层级差异，暗示后续需要更强的手段
- **不出现何三七**：第4章何三七不露面，留给第5章
- **民俗细节**：大勇学秦磊的简化仪式但撒米方式不对，暗示"只学了表面"——呼应爷爷"只学到皮毛"的主题
- **铜钱伏笔**：秦磊摸钥匙时再次感觉铜钱硌手，但这次没有在意
