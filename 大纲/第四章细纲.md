# 《我在03年炒凶宅》第四章细纲

## 第四章：门缝

### 核心目标
- 秦磊第一次直面"那里有东西"——门缝下的光被挡住
- 这是最核心的恐怖画面：有光→光被挡住→有东西站在门外→光回来→走了
- 逼出他必须面对的现实：这不是水管问题，不是隔音问题
- 章末：他想起爷爷

### 字数
2500-3000字

### 场景拆解（3个节拍）

**节拍1：不敢睡的夜晚（约600字）**

场景：又一天夜里
- 秦磊躺在床上没睡，盯着卧室门
- 门开着（第三章结尾他说再不关门了），客厅的灯也开着
- 他在等，也不知道等什么
- 前几天的事——水龙头、睡地上、敲门——他一件一件地在脑子里过
- 他开始怀疑自己是不是精神衰弱，要不要去看医生
- 但每次快说服自己"没事"的时候，就想起老大爷那句"走了半个月才被发现"
- 半个月。一个人在屋里。十五天。

**情绪：** 疲惫、焦虑、自我怀疑

**节拍2：门缝下的光（约1200字）**

场景：深夜两三点
- 他还是没睡，侧躺着，面朝卧室门
- 卧室门开着，客厅灯亮着，光从门缝底下透进来——一条细细的光线，打在地板上
- 他盯着那条光线
- 然后，光线暗了一下
- 不是灯闪了——是有什么东西经过了客厅，挡了一下光
- 他屏住呼吸，没动
- 光线又被挡住了。这次不是一闪——光没了
- 门缝底下，那一条光，被人堵上了
- 有什么东西站在卧室门外
- 他整个人僵了，手攥着被角，攥得出汗
- 不是听到什么，不是看到什么——是那条光没了，说明门外面站着东西
- 一秒、两秒、三秒……他数着，心跳一下一下砸在耳朵里
- 大概过了十秒，也可能是三十秒——他分不清，时间变得很慢
- 光回来了
- 那个东西走了
- 他没有起来去看，不敢。他蜷在床上，攥着被子，一直到天亮

**情绪：** 纯粹的恐惧，从骨头缝里渗出来的，不是吓一跳

关键细节：
- 整段不写"鬼"字，不写"什么东西"，只写光——有光、没光、光回来了
- 秦磊的生理反应：攥被角、出汗、心跳、僵住——不写"害怕"这个词
- 时间变慢的主观感受：十秒像三十秒
- 他没有起来，这不是恐怖片里那种"作死"的主角，他不敢动

**节拍3：天亮了（约700字）**

场景：第二天早上
- 天亮了他才敢动，浑身酸得像跑了一场
- 他下床，走到卧室门口，往外看——客厅空空的，灯还开着，一切正常
- 他走到厨房——水龙头没开，没滴水
- 他走到大门——锁着，链子挂着
- 一切正常。好像什么都没发生过
- 他坐在沙发上发呆，脑子里一直转
- 他在想一件事：如果只是水管问题，如果只是隔音差，如果只是自己太累了——那门缝下面的光，是什么挡住的？
- 他没法回答这个问题
- 然后他突然想起一个人——爷爷
- 不是想起来了，是脑子里自己冒出来的。爷爷坐在院子里，蒲扇拍着腿，一边喝茶一边说话的样子
- 爷爷说过什么来着？关于房子里死过人的……
- 模糊了，记不太清了。但他记得爷爷说的时候，语气很平淡，像在讲一件很普通的事
- "住进去之前，得先打个招呼。人走了不打紧，没送走才留东西。"

**结尾句/段落：**
- "我坐在沙发上，看着客厅，看着厨房，看着那扇我再也不敢关的卧室门。爷爷当年到底还说了什么？我使劲想，想得头疼，但只记得那一句。"
- 钩子：他开始主动回忆爷爷的话了——从被动承受转向主动寻找答案

**情绪：** 从恐惧转向困惑，再转向一丝微弱的希望——爷爷也许说过解决的办法

### 去AI味检查清单
- [ ] 门缝光这段绝对不用形容词堆砌，只写光有/光没
- [ ] 秦磊不会说"我害怕"，只写动作和生理反应
- [ ] 爷爷的话要像老人闲聊，不像教科书
- [ ] "人走了不打紧，没送走才留东西"——这句话要轻描淡写，不是郑重其事
