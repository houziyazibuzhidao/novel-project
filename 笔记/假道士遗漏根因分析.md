# 假道士环节遗漏根因分析

> 2026-06-13

## 问题

第4章写作时，何三七（假道士）的完整情节被遗漏。我补写时还写错了（写成"城东庙里骗子跑了"），与人物卡设定严重不符。

## 根因：细纲与人物卡的信息断裂

### 1. 细纲没有包含何三七

第4章细纲明确写了：

> **不出现何三七**：第4章何三七不露面，留给第5章

这条指令的**本意**是"何三七不在第4章正面出场"——即他不以活人身份走进场景。但被理解成了"第4章完全不需要涉及何三七的任何信息"。

实际上，人物卡和故事线总纲里，何三七的翻车是**大勇讲述的经过**的一部分——不是何三七正面出场，而是大勇转述"我请了个道士，结果他倒地上了"。这跟"何三七不露面"并不矛盾，但细纲没区分。

### 2. 故事线总纲与细纲的层级脱节

故事线总纲里，第二凶宅Arc的节拍是：
> 大勇的房子出了更凶的问题 → **请何三七翻车** → 秦磊回老家找残卷 → 处理大勇房子 → 三人组队

但细纲只覆盖了第4章（"去大勇婚房+听到脚步声"），把"请何三七翻车"推到了第5-6章。细纲节拍与总纲节拍不一致时，写作时只看了细纲，没回查总纲。

### 3. 人物卡信息没有被"强制加载"到写作流程

当前写作流程：
1. 读细纲 → 2. 构建prompt → 3. V4 Pro写初稿 → 4. 精修

问题在于：**细纲是写作的直接指令，人物卡是背景参考**。当细纲没提何三七时，V4 Pro不会主动从人物卡里"挖掘"遗漏情节。而作为编辑的我，在精修阶段也没有回查人物卡中大勇婚房的完整经历。

### 4. Kimi校对也漏了

Kimi的review关注的是：
- 人物卡一致性（职业/收入/称呼等）
- 跨章逻辑（铜钱位置/水龙头衔接等）
- 文笔和闪光点

但Kimi的校对清单里没有"对照故事线总纲检查情节完整性"这一项。它检查的是"已有内容是否正确"，不是"是否有内容被遗漏"。

## 修复措施

### 写作流程改进

1. **写每章前，强制回查故事线总纲+人物卡对应段落**
   - 细纲是节拍安排，总纲是情节完整性底线
   - 细纲说"不出现何三七"≠ 细纲说"不提及何三七事件"
   - 新增检查项：该章涉及的人物，其人物卡中的关键经历是否都在正文中有交代

2. **细纲写作规范改进**
   - "不出现"要区分：不正面出场（可在对话中提及） vs 完全不涉及
   - 每章细纲末尾加"本章应提及但不出场的事件/人物"清单

3. **Kimi校对清单补充**
   - 新增：对照故事线总纲，检查该章对应Arc的节拍是否完整
   - 新增：该章涉及人物的完整经历线，是否有未交代的节点

### 本轮修复

- 第4章假道士环节已按人物卡重写（何三七青云子正面特征：蓝布道袍/旧皮箱/铜铃/桃木剑/摆坛做法→翻车倒地昏迷→送医院）
- 细纲的"不出现何三七"改为"何三七不正面出场，但大勇转述其翻车经过"
- 人物卡中"磊子"称呼已删除，统一为"老秦"
