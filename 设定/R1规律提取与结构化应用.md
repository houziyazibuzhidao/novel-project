## 一、R1五条可复用规律

### 规律1：气氛速写（5字以内概括空间情绪）
**表现**：R1进入新场景时，能用极短的主观判断句压缩整个空间的氛围
- 原文：「像这屋子在等人。」
- 原理：不描写环境细节堆砌氛围，而是角色第一直觉直接定性
- 适用位置：**每进入一间新凶宅的第一次环顾**
- 结构位置映射：第1章进601、第4章进大勇婚房、第7章进下一间凶宅……每次"第一眼"场景

### 规律2：硬撑式恐惧（NPC行为替代心理描写）
**表现**：男性角色恐惧时不直接写"害怕"，写他故意做与恐惧相反的事
- 原文：「但他故意把声音放得很大，像是要让整个屋子听见——我不怕。」
- 原理：底层男性自尊驱动的行为——越怕越要证明不怕，声音/动作/姿态都在"表演镇定"
- 适用位置：**大勇所有出场场景、后期秦磊逐渐成长后的恐惧表达**
- 扩展：陈晓慧的恐惧方式可以相反（直接沉默/回避），形成男女恐惧表达对比

### 规律3：已知元素翻新解读（仪式结果的二次解释）
**表现**：把读者已经接受的一个结论翻过来，赋予新含义
- 原文：「灭了说明它在听。听完还不走，就是不怕你。」
- 原理：前文已建立"蜡灭=走了"的认知，R1将其翻转为"蜡灭=它听了但不怕你"，认知颠覆带来恐惧升级
- 适用位置：**每个Arc的中点转折**——第1Arc是蜡灭翻新，第2Arc可以是"送走了但东西跟着人走了"，第3Arc是"它不是一个人"
- 结构位置映射：每5-6章一个Arc，Arc中点（第3章/第8章/第13章……）需要这种认知翻转

### 规律4：物证式结尾（物理细节暗示超自然存在）
**表现**：章末不写角色感受，写一个客观物理变化
- 原文：「原来开着的窗户，不知道什么时候关上了。」
- 原理：角色坐到天亮是主观经历，窗户关上是客观证据——读者和角色一起意识到"它真的来过"
- 适用位置：**每章结尾**，替代"坐到天亮"/"一夜没睡"等收束
- 变体：门开了/灯灭了/水龙头开了/东西移位了/铜钱变热了

### 规律5：呼吸间隙中的微行为（角色无意识动作暴露内心）
**表现**：在对话和叙述的间隙插入角色无意识的身体反应
- 原文：关灯又开灯（大勇去厕所回来）、手指关节发白（端杯子）、搓手不看我
- 原理：这些动作不是"害怕"的象征，而是"害怕但不想承认"的证据——比直接写恐惧高级
- 适用位置：**所有NPC对话场景的节奏调节**，每段对话插入1-2个微行为，替代"他很紧张""声音发抖"等AI味描述

## 二、R1三版共性与差异对比

| 维度 | R1测试版（带风格prompt） | R1自由版（无风格prompt） | V4 Pro |
|------|------------------------|------------------------|--------|
| 字数 | 3656 | 2916 | 2902 |
| 设定错误 | 铜钱+瞎子二婶+年数 | 铜钱+年数 | 无 |
| 气氛速写 | ✓有 | ✓有 | ✗无（融合版已补） |
| 硬撑恐惧 | 部分 | ✓完整 | ✗无（融合版已补） |
| 翻新解读 | ✓完整 | ✓完整 | ✗无（融合版已补） |
| 物证结尾 | ✗坐到天亮 | ✓窗户关上 | ✗坐到天亮（融合版已改） |
| 信息过载 | 呼吸声+叫名+脚步 | 呼吸声+叫名+脚步 | 克制：仅脚步声 |
| 提前泄露 | 瞎子二婶名字 | ✗无 | ✗无 |
| 对话节奏 | 较冗长 | 适中 | 精炼 |

**结论**：R1带风格prompt时更冗长且容易跑偏（泄露瞎子二婶）；R1自由发挥时闪光点最集中但设定错误依旧。V4 Pro骨架最稳，缺的是R1这5条规律提供的血肉。

## 四、R1规律Prompt化（写入V4 Pro写作指令）

### 在每章写作prompt中加入以下指令块：

```
【R1规律指令】
1. 场景进入时：用一句5字以内的主观判断句压缩空间氛围，不堆砌环境描写
2. 男性角色恐惧时：写他故意做与恐惧相反的事（放大声/主动站前面/故意不关灯），绝对不写"发抖/缩成一团"
3. 每5-6章需一次认知翻转：把前文已建立的结论翻过来赋予新含义
4. 章末：用一个客观物理变化（门/窗/灯/水/物品移位）替代主观感受收束
5. 对话间隙：每段对话插入1-2个角色无意识身体反应（关节发白/关灯又开/搓手），替代"他很紧张"
```

## 五、R1写作任务规划

### 可让R1执行的结构性任务

基于R1的5条规律，R1最适合写的是**特定位置的关键段落**而非整章：

1. **每间凶宅的"第一眼"场景**（规律1）：给R1空间描述和角色状态，让它产出气氛速写
2. **NPC恐惧表达段落**（规律2+5）：给R1NPC性格和恐惧原因，让它产出行为细节
3. **认知翻转段落**（规律3）：给R1当前已建立的认知，让它产出翻新解读
4. **每章结尾**（规律4）：给R1场景状态和恐怖事件，让它产物证式结尾

### R1任务模板

```
你是一个恐怖小说的场景写作专家。根据以下信息，写一段[场景类型]：

【角色状态】...
【当前认知】...
【空间描述】...
【需要产出的内容类型】（四选一）：
  A. 气氛速写（5字以内主观判断句）
  B. 恐惧行为（硬撑式，不直接写"害怕"）
  C. 认知翻转（把"XX"翻过来）
  D. 物证结尾（客观物理变化）

要求：
- 不要写整章，只写指定段落
- 不要编造设定，严格按提供信息写
- 口语化，去AI味，2003年时代质感
```
