# 《我在03年炒凶宅》第6-10章细纲

> 基于故事线总纲：第6-9章为第二凶宅Arc收尾（回老家→拿残卷→处理大勇房子→三人组队），第10章为第三Arc开头（第一单生意）
> 
> 写作方法：R1自由版 + V4 Pro风格版 → 融合定稿
> 时代词库：每章1个方言词+1-2个时代锚点，不堆砌
> 方言原则：句尾语气词显味（噻、嘛、咯、哟），每章最多1-2个，不堆

---

## 第六章：秦家沟

**功能**：回老家——路途一笔带过→夜路回村（着重老旧阴暗的城乡面貌）→老宅→情感冲击
**字数目标**：3000-3500字
**视角**：秦磊第一人称

### 节拍

1. **路途**（~300字，一笔带过）
   - 绿皮火车三十多个小时硬座，省城转大巴，盘山路晃了四个小时到镇上
   - 车窗外的变化：城市的灰楼→县城的破街→盘山路两边全是山，手机信号一格一格地掉
   - 秦磊脑子里全是601和大勇房子的事。爷爷讲过的碎片冒出来——"没走的人""来串门的""跟回来的"——当年当故事听，现在一个比一个真
   - 到镇上天黑了。摩的师傅一听秦家沟："那地方没公路，走上去得两个多小时。"秦磊说走上去
   - 时代锚点：非典刚过有人还戴口罩 + 列车员"啤酒饮料矿泉水"

2. **夜路回村**（~800字）**← 着重描写**
   - 没等摩的，自己走。月亮还行，路看得见
   - 镇上的街：一条主路，两边是灰扑扑的门面房，卖农药的、卖种子的、诊所、照相馆，招牌一半掉漆。过了最后一盏路灯，就是黑
   - 山路窄，两边苞谷地和灌木。2003年的中西部农村——没有路灯，没有手机导航，靠月亮和脚感
   - 重点描写：老、旧、阴暗的感觉。路边的土墙房子塌了半边，里面黑洞洞的。电线杆歪着，线断了耷拉在地上没人修。河沟里的水声，远处山上有几点光——不是灯火，是别人家的电视从窗户漏出来的蓝光
   - 安静，只有虫子和自己的脚步声。偶尔有狗叫，很远
   - 秦磊想起小时候跟爷爷走过这条路，晕车吐了一路，爷爷背着他走了两个小时
   - 爷爷走了六七年了，他一次都没回来过。不是不想，是不敢
   - 远远看见秦家沟的灯火——稀稀拉拉几户人家，比记忆中又少了
   - 气氛速写（5字以内）

3. **老宅**（~1200字）**← 核心**
   - 到老宅门口。四间房倒了一间半，荒草长到膝盖。门锁生锈
   - 秦磊站门口没动。风从塌了的那间吹过来，带着一股潮气和霉味
   - 进屋——爷爷住的那间还算完整，但积了厚厚的灰。床还在，墙上挂着奶奶的遗像和一张发黄的毛主席像
   - 他开始收拾。扫灰、开窗、铺床。动作机械，不敢细想
   - 翻到爷爷床头柜——里面有个铁皮盒子，打开是几颗糖，早化成一块了。爷爷生前爱藏糖，说"苦了来一颗"
   - 秦磊坐在爷爷床上，拿着那块化掉的糖。六七年了。糖化了，屋子塌了，村里人走了一大半——他连爷爷最后一面都没见着
   - 对着老宅说话（不是仪式，是跟爷爷说）："爷，我碰上事了。你以前讲那些不是故事，我现在才懂噻。"
   - 院子里的夜风把门吹得响了一下。他没怕——这是家
   - 铺完床躺着睡不着，黑暗里盯着天花板。爷爷在的时候这屋永远有盏灯，现在全黑了

4. **第二天在村里**（~800字）
   - 被鸡叫醒。村里人少，老人居多，年轻人都出去了
   - 有认识他的喊小名"石头"——"你咋回来了嘛？你爷走了之后没人住了"
   - 老宅六年没人住，能不塌？旁边张大娘叹气："你那屋子再不修，明年还得再倒一间"
   - 秦磊打听李二叔的事，村里人不太愿意多说——"那个老辈子的事了，莫提了"
   - 问了几个人，才知道瞎子二婶还在，住在李家老屋
   - 秦磊心里一动——李二叔的后人？爷爷提过李二叔"本事大，我这点东西连皮毛都算不上"
   - 物证式结尾：出门前他摸了一下门锁，锈掉了一片，露出底下冰凉的铁

### 写作要点
- **路途是过渡不是章节**——300字交代完
- **夜路着重城乡面貌**——写2003年中西部农村的老旧阴暗，不是写恐怖，是写衰败和苍凉。土墙、断线、蓝光、狗叫——这些比鬼更让人难受
- **老宅是核心场景**——六七年没回来的愧疚，爷爷不在的实感，铁皮盒子里化掉的糖
- **秦磊的情感**：不是恐惧，是愧疚+自责——六七年没回来，连爷爷最后一面都没见着
- **"这是家"**——老宅有鬼他也不怕，因为这是他的地方。跟601形成对比
- **为Ch7铺路**：打听到瞎子二婶，引出残卷线索
- **方言**：句尾语气词点味（"懂噻""回来了嘛"），一两个就够

## 第七章：瞎子二婶

**功能**：找到瞎子二婶→拿到残卷→第一次接触端公体系
**字数目标**：3500-4000字
**视角**：秦磊第一人称

### 节拍

1. **找到瞎子二婶**（~1500字）
   - 李家老屋比秦家还破，但门前的地扫得干净
   - 瞎子二婶坐在门口晒太阳。六十多岁，瘦，眼睛闭着但不是眯着——是瞎了
   - 秦磊说明来意——他爷爷是当年追着李二叔听故事的那个人，现在他碰上事了
   - 瞎子二婶安静了很久，说了一句话："你爷爷当年听我爷爷讲故事的时候，你还没出生。"
   - 她起身摸进屋里，从神龛下面摸出一个油纸包，递给秦磊
   - 残卷：毛边纸抄的，边角全蛀了，后半本有烧焦的痕迹，只保住了一半
   - 瞎子二婶说："拿着，这是你的东西。"
   - 秦磊说这是李家的书。她没接，只是说："我爷爷等了一辈子没等到。我爹等了一辈子也没等到。我等了大半辈子。"
   - 秦磊问她等谁。她没回答。
   - 微行为：瞎子二婶递书的时候手指在油纸上停了一下，像是在确认什么——然后松手了

2. **翻看残卷**（~800字）
   - 秦磊坐在李家老屋门槛上翻残卷
   - 内容是各种处理"房子不安稳"的法子——但不是系统教程，是笔记体，一条一条的
   - 每条开头写症状（"夜半叩门""叫名不应""卧中移位"），后面写法子
   - 大勇那个——叫名字的——残卷里有。但写得极简，就几个字
   - 秦磊看不太懂，但能看出关键：比爷爷那套复杂得多，不是三样东西能搞定的
   - 还有一条残卷里提到"外来的东西"和"住里面的东西"处理方式完全不同——印证了秦磊的判断

3. **离开**（~700字）
   - 秦磊想问瞎子二婶更多，但她不再说话了，坐回门口晒太阳
   - 他把残卷用衣服包好，塞进包里
   - 走的时候回头看了一眼——瞎子二婶脸朝着他的方向，像是在看，又像不是
   - 村口遇到一个老人，说了句："你爷爷当年也拿过李家的东西。但那是活的。你拿的这个，是烧过的。"
   - 秦磊问他什么意思，老人摇摇头走了
   - 物证式结尾：秦磊把残卷打开看最后一页——有半行字被烧掉了，只能看到三个字："……不可逆"

### 写作要点
- **瞎子二婶是全书最重要的人物之一**——但此刻只露一面，保持神秘
- **残卷不展开详解**——只让秦磊看到"有对应法子"和"比爷爷的复杂"，具体内容第8章处理大勇房子时才用
- **"这是你的东西"**——核心台词，不解释，留悬念
- **方言**：瞎子二婶的对话带方言语气词（"噻""咯"），秦磊在村里也自然带出来，但每处1-2个不堆
- **气氛速写**：李家老屋——一句定性
- **残卷的"不可逆"**——长线伏笔，第一卷不回收

## 第八章：大勇的房子

**功能**：秦磊带残卷回来→按残卷处理大勇婚房→过程比601凶险得多→成功
**字数目标**：3500-4000字
**视角**：秦磊第一人称

### 节拍

1. **回来**（~400字）
   - 三十多个小时火车回到小城，直接去大勇那里
   - 大勇更憔悴了——一个人守着婚房，晓慧电话一天三个
   - 秦磊把残卷拿出来，大勇翻了两页，看不懂："这玩意儿管用？"
   - "不知道。但我爷爷的东西管用过，这个比我爷爷的更全。"

2. **残卷里的法子**（~800字）
   - 秦磊对照大勇的症状翻残卷——"夜半叫名"条目
   - 残卷写法比爷爷的复杂：不是三样东西，是七样
   - 清水、白蜡烛、米（跟爷爷的一样）+ 盐、红线、铜钱、黄纸
   - 仪式也不同：不是对着空气说三遍，是"封门、引路、送走"三步
   - 秦磊看了好几遍，有些字不认识，有些步骤写得模糊（被烧掉的部分）
   - 但核心他看懂了：大勇那个是"外面来的"，不能光送，得先封门——不让它进来，再把它送走
   - "封门"用盐和红线，"引路"用蜡烛和米，"送走"用铜钱和黄纸
   - 时代锚点：秦磊去菜市场/杂货铺买这些东西——盐、红线好买，铜钱用自己兜里的，黄纸得去殡葬店

3. **执行仪式**（~1500字）
   - 傍晚开始准备。大勇帮忙摆东西，手还是在抖，但比之前好——有盼头了
   - 封门：盐撒门槛，红线绕门框三圈。秦磊念残卷上的话，半懂不懂，但声音稳
   - 引路：点蜡，撒米，从门口往卧室方向引。蜡烛火苗猛晃了一下——它在
   - 这时候恐怖升级：蜡烛没灭，但火苗往回缩，像被什么吹了一下——但屋里没有风
   - 大勇的恐惧反应：想跑但站着不动，嘴里骂"操"，手握拳头指节发白
   - 秦磊的恐惧反应：想冲过去挡在前面，腿不听使唤——但嘴里的话没停
   - 送走：铜钱放清水碗里，黄纸烧了。秦磊说完最后一句，蜡烛灭了
   - 但这次灭的方式不一样——不是被风吹灭，是火苗自己缩回去了，像被吸进去的
   - 安静了。屋里空气"轻了"——跟601那次一样，但这次更明显

4. **善后**（~800字）
   - 大勇坐在地上，腿软了，半天才站起来
   - "老秦，成了吗？"
   - 秦磊看了一眼清水碗——铜钱沉底了（601那次铜钱浮着）
   - "成了。"
   - 两人坐在客厅抽了一夜烟，不敢睡，但确实什么都没再发生
   - 天亮了。窗户还是关着的——但这次是秦磊自己关的
   - 大勇的释然："妈的，终于安静了。"
   - 物证式结尾：秦磊收拾东西，发现兜里的三枚铜钱只剩两枚了——另一枚在清水碗里。他记不清是什么时候掉的

### 写作要点
- **仪式过程是本书第一次完整展示端公法事**——比第3章的"三样东西"复杂得多
- **残卷法子的描写原则**：说步骤不说原理，秦磊自己也不懂原理，照着做
- **恐怖升级但不失控**：比601更凶，但秦磊有残卷撑着，节奏是"紧张→成功"
- **认知翻转位置**：蜡烛灭的方式不同——"被吸进去的"vs"被风吹灭"，暗示这东西更强但被克制了
- **铜钱少了一枚**：物证式收束+长线伏笔（残卷法子有代价？铜钱是消耗品？）

## 第九章：草台班子

**功能**：处理完大勇房子→探望何三七→三人聊凶宅生意→决定组队→第一卷第二Arc收束
**字数目标**：3000-3500字
**视角**：秦磊第一人称

### 节拍

1. **探病**（~600字）
   - 何三七还在医院，精神好了不少，靠在床上嗑瓜子
   - 秦磊和大勇告诉他：房子处理了
   - 何三七眼睛一亮："怎么处理的？"
   - 秦磊没细说，只说回了趟老家找到个法子
   - 何三七没追问，但若有所思

2. **何三七的算盘**（~1000字）
   - 何三七听完他们怎么处理的房子，橘子皮一扔，开始算账
   - "你们知不知道那两套房子值多少钱？"
   - 凶宅收的时候比市价低三四成，处理干净之后正常价出手，差价就是利润
   - 何三七说：他有人脉，有渠道——谁家房子不干净、谁想偷偷出手凶宅、上哪儿搞驱邪器物，他全知道
   - 大勇动心了，但没当场拍板——他有房贷，有媳妇，不能说辞就辞
   - 秦磊也犹豫——他刚处理完两套房子，算是有经验，但把这事当职业？"
   - 何三七出了个主意："先兼职干几单。成了再说。"

3. **大勇的处境**（~600字）
   - 大勇回去跟陈晓慧说了房子处理好了，但没说怎么处理的
   - 陈晓慧回来住了一晚，说"好像确实好了"
   - 婚礼倒计时还在——一个半月，装修得赶工
   - 大勇开始算账：一个月工资两三千，房贷扣完剩一千多，装修还要花钱
   - 凶宅的差价——何三七算的——一单能赚几个月工资
   - 大勇找到秦磊："老秦，要不试试？"

4. **决定**（~500字）
   - 秦磊想了两天。残卷在他抽屉里，铜钱少了一枚
   - 他又翻了一遍残卷——还有大半看不懂，但每一条都是处理"不安稳房子"的法子
   - 如果这些法子都管用……
   - 他给何三七打了个电话："行。先试试。"
   - 何三七在电话那头笑了："得嘞。我这儿正好有个线索。"
   - 物证式结尾：秦磊挂了电话，看着桌上的残卷。翻回第一页，他发现之前没注意的四个小字——"因果自负"

### 写作要点
- **何三七的"不做法事"≠退出**——他不做法事了，但做生意可以
- **三人分工明确**：秦磊看宅子出方案，大勇跑腿谈价，何三七提供线索和渠道
- **大勇的犹豫真实**：有房贷有媳妇，不是热血一冲就辞职
- **"先兼职"的过渡**：不急，试几单再决定
- **"因果自负"**：残卷的第一条警告，长线伏笔，第一卷不回收
- **陈晓慧的侧面**：大勇说"好像确实好了"——她对丈夫做的事一无所知，也不想知道细节

## 第十章：第一单

**功能**：第三凶宅Arc开始→何三七介绍第一个客户→秦磊第一次以"专业"身份看宅
**字数目标**：3000-3500字
**视角**：秦磊第一人称

### 节拍

1. **何三七的线索**（~500字）
   - 何三七约在茶馆见面，掏出一张纸条——城西有套房子，房主想出手，开价比市价低四成
   - "这房子之前住的老两口，老头先走的，老太太在屋里住了三个月，后来也走了。走了之后隔壁邻居听见里面有动静，不敢近前。"
   - 秦磊问："你怎么知道的？"
   - 何三七嗑着瓜子："我干这行十年，你以为光骗钱呢？三教九流的消息，我比谁都灵。"

2. **踩盘**（~1200字）
   - 秦磊和大勇去看房子。城西老小区，三楼，两室一厅
   - 秦磊进门前先站了一会儿——感觉了一下。没有大勇婚房那种"沉"，但有一种"闷"
   - 气氛速写（5字以内）
   - 进屋：家具还在，老两口的遗物没清。电视、沙发、茶几上的搪瓷杯、墙上的老挂历——停在2002年
   - 卧室：两张床并排放，老头那侧的被子还叠着，老太太那侧的被子掀开着，像刚起来
   - 秦磊翻残卷对照——"住里面的"，跟601一样，不知道自己走了
   - 但他发现一个问题：残卷里"住里面的"也有细分——有的是"不知道走了"，有的是"知道走了但不愿走"
   - 这套房子——秦磊感觉是后者

3. **报价**（~600字）
   - 房主是个中年男人，老两口的儿子，在外地工作，房子空着不敢来
   - "我爸妈的东西我都没敢动。邻居说晚上有动静，我也不想管了。谁要，便宜出。"
   - 大勇跟他谈价——比何三七说的四成还低一点。房主急着脱手，没怎么还价
   - 秦磊在旁边不说话，但心里在算：这套收进来，处理干净，出手，差价能顶他半年工资

4. **准备**（~700字）
   - 回去之后秦磊翻残卷——"知道走了但不愿走"的法子比"不知道走了"复杂
   - 不是送走，是"劝走"——得知道她不愿走的原因
   - 残卷上写了几种常见原因和对应法子，但这一条又被烧掉了一部分
   - 秦磊能看懂的部分：得跟她"说"——不是像601那样说"你走了"，得说点别的
   - 但说什么？残卷没写完
   - 秦磊盯着那半行被烧掉的字，想起瞎子二婶说的"这是你的东西"——她的意思是不是，这书该由他补完？
   - 物证式结尾：秦磊把残卷合上，发现封底的内页有字——之前被油纸包着没看到——是李二叔的笔迹，只有一行："有缘人，往下写。"

### 写作要点
- **第三Arc的恐怖逻辑不同于前两个**：601是"不知道走了"（好送），大勇是"外面来的"（得封门再送），这单是"知道走了但不愿走"（得劝）
   - 这三种类型构成秦磊入行的三个台阶，难度递进
- **大勇开始发挥作用**：谈价是他的强项——跑建材业务的嘴皮子
- **残卷是活的**：不是固定教材，是笔记+待续——"有缘人往下写"意味着秦磊需要自己补
- **秦磊的贪财vs善良**：他算差价时是贪的，但进屋看到老两口的遗物时心软了——这个人做不了纯粹奸商
- **时代锚点**：茶馆+嗑瓜子+搪瓷杯+老挂历+2003年房价
