# R1风格特征提取与V4 Pro复刻方案

> 用途：在R1（deepseek-reasoner）2026-07-24下线前，将其不可替代的文学风格固化为可程序化的prompt指令，供V4 Pro（deepseek-v4-pro）复刻。
> 适用范围：本项目《我在03年炒凶宅》后续章节（第4章起），及其他民俗恐怖小说项目。

---

## 一、R1不可替代的5个核心优势

经对比测试，以下5项是R1的"硬指标"——V4 Pro初稿无法自动达到，必须prompt化后才能复刻：

| # | 优势维度 | 具体表现 | 复刻难度 |
|---|---------|---------|---------|
| 1 | **钩子工程** | 首句/段落的悬念密度、读者"不得不翻页"的强迫感 | 高（需prompt精确拆解） |
| 2 | **留白叙事** | 恐怖不描写、不解释、让读者自己脑补 | 高（需反向指令：禁说破） |
| 3 | **数字痛感** | 底层攒钱的具体数字带来的扎心感 | 中（需数字模板） |
| 4 | **暧昧台词** | 老大爷/中介只说一半，暧昧感拿捏 | 中（需半句模板） |
| 5 | **结尾回弹** | 章末钩子干净、不泄气、有余韵 | 中（需结尾结构模板） |

---

## 二、5项优势的具体拆解与prompt化

### 1. 钩子工程 → "首句三段式"模板

**R1原版（第1章）：**
> 后来很多人问我，你当时怎么敢买那个房子的。
> 我说，便宜啊。
> 其实不只是便宜。

**拆解为可复刻的结构：**

```
【钩子模板·三段式】
第1句：后置视角的疑问（"后来很多人问我……"）——制造时间纵深
第2句：主人公的直接回答（"我说，便宜啊。"）——简单到让读者好奇"就这？"
第3句：转折+否定（"其实不只是便宜。"）——把读者钉住，必须往下看

限制条件：
- 三句合计不超过30字
- 第2句必须是主谓宾完整句，不能是形容词
- 第3句必须含"不只是/不全是/不止"等否定词
```

**V4 Pro复刻prompt示例：**
> 本章开头使用"钩子三段式"：第1句用"后来很多人问我/后来有人问我/后来有人好奇"开头，第2句是主人公的直接回答（不超过5个字），第3句用"其实不只是/其实不只是/不全是"转折。三句合计不超过30字，直接进入叙事，不解释。

---

### 2. 留白叙事 → "三不原则"反向指令

**R1原版（第1章周哥不敢进屋）：**
> 他站在那儿，手攥着钥匙，犹豫了一下。然后他把钥匙递给我。
> "你自己进去看吧。"
> "你不进去？"
> "我……就不进去了。"他笑了笑，那笑不太自然，"我在这等你。"
> 我当时没多想。中介嘛，有的就是懒，能少走一步是一步。

**R1没写什么（这才是恐怖的来源）：**
- 没写"周哥害怕"
- 没写"房子有问题"
- 没写"我感觉到不对劲"
- 只写了"笑不太自然"和"我"的自我解释

**拆解为"三不原则"：**

```
【留白模板·三不原则】
1. 不直接描写恐惧：禁用"害怕/恐惧/惊慌/不安/心里发毛"
2. 不直接解释异常：异常发生后，主人公必须给自己找理由（老房子嘛/水压问题/记性差）
3. 不直接描写鬼：不写鬼的样子、动作、意图，只写"东西"存在过的痕迹（香梗歪了、锅盖翻了、被子叠了）

替代写法：
- 不写"我害怕" → 写"我盯着天花板，没闭眼"
- 不写"有鬼" → 写"茶几上的碗，三根香梗还在"
- 不写"房子有问题" → 写"周哥把钥匙递给我，自己没进门"
```

**V4 Pro复刻prompt示例：**
> 恐怖描写遵循"三不原则"：不直接写恐惧心理（禁用害怕/恐惧/惊慌），不直接解释异常（主人公必须给自己找理由），不直接描写鬼（只写异常痕迹：物品移动、声音、光线变化）。异常发生后，主人公的第一反应是"找说法糊弄自己"。

### 3. 数字痛感 → "数字三连"模板

**R1原版（第1章攒钱）：**
> 打开存折那天，我看着上面那四万二，说实话有点想哭。加上年终奖和手头零碎，总共四万八出头。
> 那时候小城房价虽然不比大城市，但好的地段也得两千多一平。四万八，买个卫生间都够呛。

**拆解为"数字三连"结构：**

```
【数字痛感模板】
第1个数字：主人公的积蓄（具体到个位数，带情感反应"有点想哭"）
第2个数字：对比物的价格（房价/物价，让第一个数字显得渺小）
第3个数字：主人公的应对/妥协（"五万二"——差四千，借钱，缝扣子）

限制条件：
- 三个数字必须在同一段或相邻两段出现
- 数字之间要有"落差感"（四万八 vs 两千多一平）
- 必须伴随一个具体动作（打开存折、翻通讯录、打电话）
```

**V4 Pro复刻prompt示例：**
> 写攒钱/经济状况时，使用"数字三连"：先写主人公的具体存款（带情感反应），再写对比物的价格（让存款显得渺小），最后写主人公的妥协或差距填补方式。数字要具体到个位数，伴随具体动作（打开存折、翻钱包、打电话）。

### 4. 暧昧台词 → "半句+省略"模板

**R1原版（老大爷爆料）：**
> "那阵子，楼道里总有一股子味。"老大爷说着，用拐杖指了指楼道，"那味挺冲的，一开始以为是谁家死猫死狗了……"
> 他站在原地，没动。
> "楼里夏天蚊子多，大家点蚊香……"老大爷压低了声音，"也不全是蚊子的事。"

**拆解为"半句+省略"结构：**

```
【暧昧台词模板】
1. 信息说一半："死了。半个月后才被人发现。"——说完停顿，不展开
2. 转移话题后再折回：先说"蚊子多"，压低声音说"也不全是蚊子的事"
3. 用身体语言代替解释："老头咳了一声，摇了摇头，又点了点头"——动作矛盾，信息不确
4. 结尾不追问：主人公听完只说"哦""不怕"——不追问，让信息悬在空中

关键规则：
- NPC每段话不超过2个信息点
- 必须有至少1处停顿（……）或1处动作打断（摇头/咳嗽/笑）
- 主人公不能追问，必须用"自我解释"消化信息
```

**V4 Pro复刻prompt示例：**
> NPC爆料台词使用"半句+省略"结构：每次只说1-2个信息点，说完停顿（……）或动作打断（摇头/笑/咳嗽），信息不展开。主人公听完不追问，用"哦"/"不怕"/自我解释消化。关键信息用压低声音、旁敲侧击的方式给出。

### 5. 结尾回弹 → "尾句反抛"模板

**R1原版（第1章结尾）：**
> 又快要睡着的时候——
> 滴答。
> 我一下坐起来。
> 我关灯前明明关紧了。

**拆解为"尾句反抛"结构：**

```
【结尾钩子模板】
1. 先放松："又快要睡着的时候"——让读者以为安全了
2. 再打破："滴答"——短促的声音，打破安静
3. 主人公反应："我一下坐起来"——动作代替心理
4. 尾句反抛："我关灯前明明关紧了"——不说害怕，说"明明"——暗示不可能，但发生了

限制条件：
- 结尾最后一句必须是陈述句，不是感叹句
- 最后一句必须含"明明/本来/已经/怎么又"等"不应该"的暗示词
- 结尾不超过50字，干净收束
```

**V4 Pro复刻prompt示例：**
> 章末钩子使用"尾句反抛"：先写放松（快要睡着/刚松口气），再写打破（声音/异常），主人公动作反应（坐起来/睁开眼），最后一句用"明明/本来/已经"暗示"不可能，但发生了"。结尾不超过50字，不用感叹号。

## 三、V4 Pro复刻的完整prompt框架（每章开头喂一次）

```
【角色卡】（完整20000字，不传精简版）
【民俗资料库】（相关章节节选，约3000字）
【前文摘要】（上章结尾300字，确保衔接）
【本章细纲】（节拍+字数+钩子位置）

【写作风格指令·R1复刻版】
1. 开头：使用"钩子三段式"（30字以内，后置视角+直接回答+转折）
2. 恐怖：遵循"三不原则"（不直接写恐惧/不解释异常/不描写鬼）
3. 经济：使用"数字三连"（存款→对比→妥协，具体到个位数）
4. NPC：使用"半句+省略"（信息说一半，动作打断，主人公不追问）
5. 结尾：使用"尾句反抛"（放松→打破→动作→"明明"收束，50字内）
6. 秦磊声音：24岁男生，务实，给自己找理由，不抒情，口语化（他妈的/我擦/操蛋）
7. 恐惧反应：愤怒+想冲出去+身体不听使唤，禁用发抖/脊背发凉/毛骨悚然
8. 2003年质感：诺基亚/发短信/存折/泡面/19块9/雪花点电视/拔插头
9. 去AI味：禁用"令人毛骨悚然/不寒而栗/脊背发凉/毛骨悚然/诡异/令人窒息"
10. 民俗：仪式简化版（清水+蜡烛+米），爷爷只学到皮毛，秦磊只记得碎片

【输出要求】
- 每章2000-3000字
- 章末必须有钩子
- 第1人称，秦磊视角
- 短句为主，段落不超过5行
- 恐怖从日常中渗出来，异常之间夹正常（上班、吃饭、打电话）
```

## 四、R1下线前建议Coze做的3件事

### 1. 用R1做"风格提取实验"（推荐立即执行）

用同一个prompt分别调R1和V4 Pro写第4章开头（300字），对比两者的差异，记录R1独有的表达方式。这个对比样本可以进一步细化上述5项优势。

**具体操作：**
- Prompt：只给人物卡+"写第4章开头，秦磊接到大勇电话后去大勇婚房"，不给风格指令
- 对比维度：钩子、留白、数字、台词、结尾
- 记录R1独有的句子，加入本模板

### 2. 用R1写第4章初稿（如果时间允许）

第4章是"第二凶宅"（大勇婚房）的开端，是全书的第一个转折点。如果R1能写出一版高质量的初稿，可以作为V4 Pro后续章节的"风格基准"。

**原因：**
- 第4章需要新的钩子（不同于第1章的"买房"钩子）
- 第4章需要新的人物（陈晓慧首次出现，何三七即将出场）
- 第4章的文学性定调会影响后续20+章

### 3. 把R1风格模板固化为项目文档

将本文档存入 `设定/写作风格模板.md`，作为全书标准。后续V4 Pro写每一章时，都按这个模板喂指令。

## 六、R1 vs V4 Pro 成本对比（供决策参考）

| 项目 | R1 | V4 Pro | 建议 |
|------|-----|--------|------|
| 上下文 | 128K（需精简） | 1M（完整传入） | V4 Pro完胜 |
| 人物卡遵守 | 经常偏离，需精修 | 基本遵守，微调即可 | V4 Pro完胜 |
| 文学性（钩子/留白） | 好，但不可控 | 稳，需prompt化 | R1略胜，但可复刻 |
| 精修工作量 | 大（逐段改） | 小（局部调） | V4 Pro完胜 |
| 成本 | 较高 | 缓存命中后极低 | V4 Pro完胜 |
| 退役时间 | 2026-07-24 | 长期可用 | V4 Pro完胜 |
