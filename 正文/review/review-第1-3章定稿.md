# 校对反馈：第1-3章定稿（最新重构版，2026-06-12）

## 一、整体评价

3章重构节奏紧凑，恐怖递进（水龙头→睡地上→敲门→门缝光→回忆爷爷→蜡烛仪式）弧线完整。去AI味良好，口语感强。新版恐怖细节比旧版更狠：被子叠好、新香梗凭空出现、水龙头三次滴答递进、门缝光被挡住两次。结尾大勇电话钩力强。但**人物卡设定多处被改写，需要修正**。

## 二、🔴 必须修正：人物卡设定矛盾

### 1. 职业：超市理货员 → 销售五金（本科营销）

| 人物卡（设定） | 新版正文（第一章） | 影响 |
|--------------|-------------------|------|
| 大学本科，市场营销专业 | 未提及 | 切断学历线 |
| 销售五金配件，底薪800+提成 | "在超市当理货员，一个月也就七八百" | 切断营销底子线 |
| 懂包装和话术 | 无 | 后续凶宅买卖缺乏能力来源 |
| 攒了两年 | "攒了四年" | 时间线拉长，金牛效率变低 |
| 四万二+年终奖=四万八 | "两万三" | 金额腰斩，后续与"贪财"动机不匹配 |
| 两室一厅60平 | "一室一厅" | 切断"出租另一间"计划 |
| 601 | "602" | 房号变更 |

**修正建议：**
- 第1章："我虽说24岁，在**建材市场做销售**，一个月底薪八百，提成好的时候能拿一千出头，攒了两年才存下四万二"
- 保留"借我妈四千"，总价四万六→五万二差额可以补一句"加上年终奖和零碎，凑够了"，或者留一个"还差六千"的悬念
- 房型改回"两室一厅"，"六十平，客厅不大但够用，自己住一间，另一间打算租出去"
- 房号确认：601还是602？人物卡旧版是601

### 2. 数字问题：两万三 + 四千 = 两万七 ≠ 五万二

**严重数学错误。** 如果总价是五万二，两万三+四千只够付一半不到。需要确认：

- 方案A：总价写错了，应该是两万七？（但"五万二"是标题，不太可能改）
- 方案B：两万三是首付，其余贷款？（但正文写"一次性付清"）
- 方案C：两万三只是秦磊自己攒的，爹妈或其他人也出了？（正文说"加上我妈那儿还有点"，但只提了四千）
- 方案D：笔误，两万三应为四万二或四万三

**建议修正：** 改为"我自己攒了四万二，加上我妈四千，五万二，刚好"，或"我自己攒了三万八，加上我妈四千，四万二，首付够了，剩下分期"——但"一次性付清"和"五万二"需要同时满足。最合理的是秦磊手头四万二+借妈四千+年终奖/零碎凑够五万二。

### 3. 房型：一室一厅 → 两室一厅

人物卡明确"两室一厅，60平"。新版写"一室一厅"，且卧室"十来平米"，客厅"不大"——加起来没有60平。这不仅影响空间感，更影响后续剧情（秦磊计划"另一间租出去"）。

**修正建议：** 改回"两室一厅，六十平，客厅不大，主卧朝南有个十来平米，次卧小点，放张床就满了"。

### 4. 房号：602 → 601

旧版（人物卡/起源线）一直写601。新版写602。如果刻意变更，需确认是打字错误还是设定变更。602是另一户，可能影响老大爷对话（"对门"vs"隔壁"）。

**建议修正：** 统一为601，与旧版一致。除非有特别理由（如601是老太太，602是秦磊？但正文说"那屋之前住的老太太"，暗示秦磊住的正是老太太那屋）。

## 三、🟡 建议修正：细节与伏笔

### 5. 周哥敢进屋 → 丢失"不敢进屋"核心伏笔

旧版中周哥"不敢进屋"是全书第一个让读者知道"这房子有问题"的暗示。新版中周哥直接进屋，带看房，解释香梗，这个**核心悬念钩子被彻底删除**。

**建议修正方案：**
- 方案A：保留新版周哥进屋，但加一个反常细节——比如周哥全程站在门口不进卧室，或者看房时一直看表、催着走，离开时长舒一口气
- 方案B：恢复旧版周哥不敢进屋，只在新版中加入老大爷更早出场（如看房时就碰到）
- 方案C：周哥进屋但香梗/红纸条处他明显心虚，说"我让人收拾过了"时眼神闪躲——用"掩饰"替代"不敢进屋"，暗示他知道内情但不想说

目前方案C最自然，只需要加一点周哥心虚的描写。

### 6. 大勇：首次出场太晚，且结婚时间矛盾

**问题A：** 新版大勇只在第三章末尾电话出场。人物卡设定大勇和秦磊是"大学同学，上下铺住了四年"，"毕业后留在同一座城市"。新版完全没有展现这段关系，读者第三章才知道有大勇这个人，缺乏铺垫。

**建议：** 第二章可以提一句大勇——比如秦磊收到短信或想起大勇，或者水龙头第一次滴答时想"这事要是告诉大勇，他肯定说我神经病"。不需要大篇幅，但需要让读者知道秦磊有这样一个朋友。

**问题B：** 第三章说"大勇去年结的婚，媳妇是隔壁县的"。但人物卡设定：大勇2003年"到了谈婚论嫁的阶段"，"女朋友陈晓慧是城里姑娘，家里条件不错"。

- "去年结婚" vs "谈婚论嫁（未婚）"——时间线矛盾
- "隔壁县的" vs "城里姑娘"——人设矛盾
- 人物卡说大勇是"1981年生"，2003年22岁，"到了谈婚论嫁阶段"但没结婚。新版说"去年结婚"（2002年），那大勇21岁就结婚了？而且陈晓慧的人设是"城里姑娘，家里条件不错，性格强势"，新版改成"隔壁县的"后，"城里姑娘"的设定和后续"惧内"来源（娘家强势）都丢了。

**建议修正：** 改回"大勇到了谈婚论嫁的时候，女朋友陈晓慧，城里姑娘，家里催得紧，正凑钱买房"。电话内容改为"我跟我媳妇，准备搬新房"或"刚买了套房准备结婚"，而不是"刚搬进来"。

### 7. 秦磊"对朋友吝啬"特质未展现

人物卡说秦磊"对朋友吝啬，但真有困难花钱从不含糊"。新版中秦磊完全没有跟朋友互动的场景（除了第三章末尾接电话）。建议第二章可以加一句秦磊想叫大勇来帮忙但"算了，叫他来还得管饭，费钱"，或者电话里大勇说"你搬新家也不叫我来喝两杯"秦磊回"请你喝酒得花钱"——用这种方式轻笔带过。

### 8. "金牛座就是这样"——星座标签生硬

第三章："金牛座就是这样，什么事都想自己扛，能不麻烦别人就不麻烦别人。"

人物卡写金牛座特质是"务实、固执、对钱有执念但不是守财奴"，没有"自己扛"这一条。"自己扛"是普遍性格，不是金牛座专属。这句话像作者跳出来解释，破坏第一人称代入感。

**建议：** 删掉"金牛座就是这样"，直接写"我从小就这德行，什么事都想自己扛，能不麻烦别人就不麻烦别人"，或者"我这人就这样，能自己扛的绝不张嘴"。

### 9. "超市大姐认识我"——如果改回五金销售，需同步改

如果职业改回销售五金，第三章买蜡烛的场景不能写"超市大姐认识我"（因为不是在超市工作了）。可以改为：
- "我下班路过一家杂货铺，挑了五根白蜡烛"——杂货铺老板不认识他
- 或者保留在超市买东西，但秦磊只是顾客，不是员工

### 10. 水龙头结尾模糊

第一章结尾："我关灯前明明关紧了。"

这句话暗示秦磊最后一次拧紧了水龙头、关了灯，然后又被滴答声吵醒。但前文写"我用手指抵着水龙头口，凉凉的，湿湿的。然后水停了。我松了口气。回屋躺下。又快要睡着的时候——滴答。我一下坐起来。"

这里逻辑是：秦磊拧紧了→手指抵着→水停了→躺下→又听见滴答。但"又快要睡着的时候——滴答"与"我关灯前明明关紧了"之间，没有交代秦磊是否第三次去拧。读者不清楚秦磊最后有没有第三次去厨房。

**建议：** 在结尾前加一句"我第三次爬起来，走到厨房，水龙头好好的，没滴水。但我明明听见了。"或者删掉"我关灯前明明关紧了"，改为"我躺下，水龙头不响了。但我刚闭上眼——滴答。又响了。"

### 11. "钥匙扣上的旧铜钱"——只出现一次，后续无回收

新版第一章："钥匙扣上有个旧铜钱，磨得发亮，不知道谁挂上去的。我后来才注意到。"

然后第二章、第三章完全没再提到这个铜钱。伏笔追踪表说"与残卷体系相关，李二叔的器物之一"，但正文没有任何暗示。如果三章内不提，读者会遗忘。

**建议：** 第二章或第三章中，秦磊摸钥匙时可以再提一句"那把铜钥匙还在兜里，钥匙扣上的旧铜钱硌着手心"，保持存在感。或者第一章不强调"旧铜钱"，只写"钥匙扣上挂着个旧东西"，等后续再揭示。

### 12. 电视开关"特别紧"——与"房子自带旧电视"矛盾

新版第二章："搬进来第一天我就发现这台破电视有个毛病——开关特别紧，关的时候需要用力按。"

旧版设定是房子自带"大屁股电视"，屏幕落灰。如果电视是房子自带的旧电视，"开关特别紧"是合理的。但如果秦磊自己没电视（搬家时只有"几件衣服、被褥、搪瓷缸子"），那电视是房子自带的。这没问题，但需要确认：秦磊对这台电视的熟悉程度是否足够？搬进来第一天就发现开关紧，说明他当天就开过电视。可以接受。
