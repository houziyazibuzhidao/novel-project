# 我在03年炒凶宅

> 中式民俗恐怖小说 · 起点中文网首发 · Webnovel出海

## 项目简介

2003年，二十四岁的秦磊贪便宜买了套凶宅，从此踏入凶宅买卖的行当。

## 目录结构

```
设定/        ← 人物卡、时间线、世界观、残卷体系
大纲/        ← 卷章规划、细纲
正文/
  raw/       ← DeepSeek R1初稿
  final/     ← 精修定稿
  review/    ← Kimi校对反馈
翻译/
  webnovel/  ← Webnovel英文版
笔记/        ← 写作日志、灵感记录
```

## 协作分工

| 角色 | 工具 | 职责 |
|------|------|------|
| 写手 | DeepSeek R1 | 出初稿 |
| 编辑 | Coze Agent (小说) | 细纲→调R1写→精修去AI味→成稿 |
| 校对 | Kimi | 一致性检查、伏笔追踪、设定校验 |
| 翻译 | Kimi | Webnovel英文版 |
| 主人 | 你 | 方向、思路、节点、最终拍板 |

## 写作规范

- 短章流：每章2000-3000字
- 章末有钩子
- 去AI味：禁套路词，加生活化小动作/感官细节
- 真鬼设定：拒绝"最后都是人做的"套路
- 第一卷名：残卷
